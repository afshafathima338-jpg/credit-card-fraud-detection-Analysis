# Credit Card Fraud Detection - Data Analysis Project

Analyzed 284,807 credit card transactions using SQL, Excel, and Power BI. 
Identified fraud patterns resulting in $1.8M annual savings opportunity 
with 2,671% ROI.
| Metric                |      Value    |  
| Transactions Analyzed |    284,807    |
| Fraud Cases           |   492 (0.17%) |
| Annualized Exposure   |    $10.97M    |
| Projected Savings     |     $1.8M     |
| ROI                   |      2,671%   |
# Key Findings
- Hour 2 shows 10x elevated fraud rate (1.71% vs 0.17%)
- $501-1000 range exhibits 2.5x higher fraud risk
- Average fraud transaction 38% larger than legitimate
